# Vim 使用教程
## 1.功能分类
1. 正常模式 --> 浏览
   
2. 编辑模式 --> 逐字添加和修改文本 
   
3. 命令行模式 --> 保存/退出/搜索

## 2.基本指令
1. 进入文件
```
#vim 文件名
vim helloworld.cpp
```
2. 进入编辑模式  
   进入文件后默认是正常模式，点击小写```i```进入编辑模式,下方显示```INSERT``` 

3. 退出文件  
   先从编辑模式按```Esc```退回正常模式，然后在正常模式输入```:```(注意为英文冒号)进入命令行模式，在冒号后输入```q```即可退出

4. 保存/不保存退出  
   ```:q!```不保存退出  
   ```:wq```保存退出

## 3.编辑指令
```i``` -插前  
```a``` -附后  
```h/j/l/l``` -分别对应左下右上  
```o``` -新增下一行  
```O``` -新增上一行  
```G``` -到最后一行  
```gg``` -到第一行
